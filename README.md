# 🤖 FAQ Chatbot (Python Project)

## 📌 Project Description
This is a simple **Python-based FAQ Chatbot** that answers frequently asked questions related to an online learning platform.  
The chatbot uses keyword matching to respond to user queries.

---

## 🚀 Features
- Answers questions about:
  - Courses
  - Duration
  - Certificate
  - Fees
  - Contact Information
- Simple command-line interface
- Beginner-friendly Python project
- Easy to run and modify

---

## 🛠️ Technologies Used
- Python 3

---

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Open terminal or command prompt in the project folder.
3. Run the following command:

```bash
python faq_chatbot.py
